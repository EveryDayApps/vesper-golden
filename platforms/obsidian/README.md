# Vesper Golden: Obsidian

Obsidian port. A full theme: app chrome, markdown reading and edit views, and code-block syntax, all in the warm gold-on-black palette. Ships dark and light variants.

Install from the community gallery: **Settings > Appearance > Themes > Manage**, search "Vesper Golden". Listing: https://community.obsidian.md/themes/vesper-golden.

- `theme/` holds the theme: `manifest.json` (name and version) and `theme.css` (every color, derived from the repo-root `palette.json`).
- Build the installable zip with `cd theme && ./pack.sh` (no network, nothing compiled).

The zip contains a `Vesper Golden/` folder. Install it by extracting that folder into a vault's `.obsidian/themes/`, then pick the theme under Settings > Appearance > Themes.

Full install and iteration steps are in [`../../docs/install.md`](../../docs/install.md).
